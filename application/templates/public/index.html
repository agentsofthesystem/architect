Agents of the System

Agents of the System is an ecosystem of free & open-source software. Deploy dedicated game servers and use our online control panel. Everything is built with Python. Designed with a distributed approach in mind, Agent Smith securely runs all dedicated game servers while The Architect provides collaborative and simple UI to end users. Join The System

What are the Agents of the System?

The Architect is a browser-based web application control panel & Agent Smith is a system tray application.
The Architect

A Cloud Hosted Control Panel for Agents which control your dedicated game servers, and also a Social Platform.

  • Agent Management

    The Architect can manage as many Agent Smith instances across you and your friend group as needed.

  • Collaboration

    Build up a network of friends and create groups. Collectively pool as many Agent Smith copies as you like and share access.

  • Automation

    The Architect's goal is to seemlessly automate the managment of dedicated game servers.

Agent Smith

Self-contained Python-Based Agent Software that can manage your dedicated game servers locally, and may also be used with The Architect.

  • Dedicated Server Management

    Installs, maintains, and provides server management capability.

  • Control

    Provide an API, Tokenized Access, and secure SSL communication to any friend you want or connect to the Architect.

  • Transformation

    Turn any PC into a potential host... See our supported games

The System

All the software tools together collectively make up Agents of The System. The end user may chose which components to use!

Why Agents of the System?

  • Free & Open Source

    Everything is 100% free to use, and whatever runs on your hardware is open source.

  • Hybrid - Scalable Distributed Cloud Approach

    Agents of the System solves dedicated server management and automation by using a distributed systems approach. Other solutions available are monolithic in nature, or require a lot of setup to achieve your goal, which is a game server control panel.

  • User Friendly

    No need to set anything up beyond Agent Smith. No special pre-requisite software to install. Simply download and run it. Written almost entirely in python and made to run self-contained. Everything else is in the Cloud. No setup required.
