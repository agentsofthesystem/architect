Why Agents of the System?

Easy...
  • Free & Open Source

    Agents Of the System is 100% free and licensed under an AGPL-v3 license. All of our code is completely open source as well.

  • Cloud Oriented

    Self-Host or use our Cloud Option, it's entirely up to you. Just want to use Agent-Smith without the Architect, that's okay.

  • Keep it Simple Stupid

    Many other tools make use of Docker, and that's great, but realistically how many people know how to set that up fast? Agent Smith allows supported game servers to be directly installed on your machine using Steam as the source.

  • User Friendly

    We don't use the shiniest Web Front End, but this tool is friend oriented and is dead simple to use. You're even free to ditch it and simply use the Agent Smith client. You become the operator!

  • Developer Friendly

    The Author respects every open source project, but wanted to build one that was as close to 99.9% Python based as possible to open the door to as many contributors as possible.
