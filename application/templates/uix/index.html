Agents of the System

Join forces with Agent Smith and The Architect to manage your Dedicated Game Servers directly on your own hardware. Make friend groups and manage your servers your way. A distributed approach to collaborative Dedicated Game Server administration. Join The System

What are the Agents of the System?

The Architect runs in your browser & Agent Smith on your PC. Stick with Agent Smith or join forces with The Architect. The choice is yours.
The Architect

A Cloud Hosted Control Panel for Agents which control your dedicated game servers, and also a Social Platform.

  • Agent Management

    The Architect can manage as many Agent Smith instances across you and your friend group as needed.

  • Collaboration

    Build up a network of friends and create groups. Collectively pool as many Agent Smith copies as you like and share access.

  • Automation

    The Architect's goal is to seemlessly automate the managment of dedicated game servers.

Agent Smith

Self-contained Python-Based Agent Software that can manage your dedicated game servers locally, and may also be used with The Architect.

  • Dedicated Server Management

    Installs, maintains, and provides server management capability.

  • Control

    Provide an API, Tokenized Access, and secure SSL communication to any friend you want or connect to the Architect.

  • Transformation

    Turn any PC into a potential host... See our supported games

The System

All the software tools together collectively make up Agents of The System.

Why Agents of the System?

  • Free & Open Source

    Agents Smith, The Operator, and all documentation are 100% free to use and open source.

  • Hybrid - Scalable Distributed Cloud Approach

    Agents of the System solves dedicated server management and automation by using a distributed systems approach. Other solutions available are monolithic in nature, or require a lot of setup to achieve your goal, which is simply running your dedicated game server.

  • User Friendly

    No need to set anything up beyond Agent Smith. No special pre-requisite software to install. Simply download and run it. Written 99% in python and made to run self-contained. Everything else is in the Cloud. No setup required. Get Started Here!
